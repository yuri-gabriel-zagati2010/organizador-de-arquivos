# Organizador de Arquivos em Python

Script que organiza automaticamente arquivos em pastas com base na extensão.

## Categorias suportadas

- Documentos
- Imagens
- Vídeos
- Áudios
- Compactados
- Instaladores
- Fontes
- Outros

## Como usar

1. Após baixar o arquivo ZIP do projeto, extraia o script em qualquer local do seu computador (exemplo: pasta Documentos ou Área de Trabalho).
2. Abra o terminal (CMD ou Windows PowerShell) no local onde o script foi extraído.
3. Execute o comando:

   python organizador.py

4. Digite o caminho da pasta que você deseja organizar e pressione Enter.

O script criará as pastas automaticamente e moverá os arquivos para seus respectivos lugares.
