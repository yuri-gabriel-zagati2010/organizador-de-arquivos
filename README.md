# Organizador de Arquivos em Python

Script que organiza automaticamente arquivos em pastas com base na extensão.

## Categorias suportadas

- Documentos
- Imagens
- Vídeos
- Áudios
- Compactados
- Instaladores
- Fontes
- Outros

## Como usar

1. Coloque o script na pasta que deseja organizar
2. Execute:

python organizador.py

O script criará pastas automaticamente e moverá os arquivos.
